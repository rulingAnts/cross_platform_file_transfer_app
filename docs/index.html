Technical Specifications

Performance

  • Multi-stream parallel transfers (1-8 streams)
  • 1 MB chunk size for optimal throughput
  • Dynamic stream adjustment
  • Up to 250 Mbps on WiFi Direct
  • <200 MB memory for 10 GB files

Security

  • TLS 1.3 encryption (AES-256-GCM)
  • Certificate pinning (TOFU model)
  • SHA-256 checksums per chunk
  • Man-in-the-middle attack detection
  • No data leaves your local network

Network

  • mDNS/Bonjour discovery
  • WiFi Direct (Android-to-Android)
  • Hotspot mode support
  • Cloud relay infrastructure (optional)
  • IPv4 and IPv6 support

Features

  • Resume interrupted transfers
  • Automatic folder compression
  • Transfer history with search
  • Battery monitoring
  • Wake lock (prevent device sleep)

Frequently Asked Questions

Is Rapid Transfer really free?

Yes! Rapid Transfer is 100% free and open source (MIT License). No ads, no subscriptions, no hidden costs.

Do files go through the internet or a server?

No. All transfers happen directly between your devices on your local network. Your files never leave your control unless you explicitly set up the optional cloud relay.

How fast are the transfers?

Speed depends on your network. On typical home WiFi: 10-60 MB/s. With WiFi Direct: up to 250 Mbps (31 MB/s). That's 1 GB in under 50 seconds!

Is it secure?

Yes. All transfers use TLS 1.3 encryption (same as banking apps). Certificate pinning prevents man-in-the-middle attacks. SHA-256 checksums ensure data integrity.

Can I transfer between different networks?

By default, devices must be on the same local network. However, you can set up the optional cloud relay for cross-network transfers (see documentation).

What file types are supported?

All file types! Photos, videos, documents, folders‚Äîanything. Folders are automatically compressed for efficient transfer.

Is there a file size limit?

No hard limit. The app has been tested with 10+ GB files. You'll get a warning for files over 10 GB, but you can still transfer them.

What if a transfer gets interrupted?

No problem! Transfers automatically resume from where they left off when devices reconnect (within 24 hours).
